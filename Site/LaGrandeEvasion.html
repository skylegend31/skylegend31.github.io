Présentation du Projet

I--Chronologie de Réalisation

Nous avons commencé par importer un personnage sur Unity et lui donner toutes les caracteristiques physiques attendues(masse,capsule de collision,...) et des capacités physiques(mouvements, saut, sprint,...). Ensuite, nous avons ajouté les animations. Nous avons aussi créé un script pour que la caméra suive le joueur. Pour l'instant en 3e personne pour la démonstration mais sera en 1ere personne dans la version finale de notre jeu vidéo. Pendant qu'une partie de l'équipe s'occupait de ça, l'autre s'est occupée de modéliser la prison sur le logiciel 3D Blender pour ensuite l'intégrer au projet Unity. On a reussi à faire réagir les portes au contact du joueur. Enfin, nous avons commencé à implémenter l'IA.


II--Logiciels utilisés

Pour la conception 3D, nous avons choisi Blender. Pour la programmation, nous avons choisi Visual Studio Code et Unity pour lier le tout.
